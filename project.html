Applicazione
Password Safer

Ho creato un app per dispositivi mobile che funge da password manager, tramite una registrazione con mail e password, l'utente sara' in grado di memorizzare le password su un database firebase con i dati crittografati.


Login e Registrazione

Password Safer

L'applicazione permette di registrare le password per non smarrirle, per fare cio' occorre registrarsi con email e password e in seguito effettuare il login. In caso le credenziali per il login andassero smarrite, e' presente un recupero password, infatti viene inviata una mail al proprietario dell'account, che puo' reimpostare la password.


Pagina Principale

Password Safer

La home si presenta con l'elenco di tutte le password dell'account in sessione presenti sul database, all'interno di essa si puo' ricercare una password tramite una barra di ricerca. Cliccando su una singola password ne si puo' visualizzare il contenuto potendo anche copiare il nickname, la password e molto altro.


Aggiunta Password

Password Safer

Ovviamente l'applicazione permette di aggiungere una nuova password tramite il simbolo del + situato nella home, l'aggiunta richiede nome, username o mail, password e sito web. I dati verranno crittografati e infine salvati sul database, in modo da tenerli segreti e visibili solo al proprietario dell' account.
